Before going to ask business question, i would like to dive deep into understanding the dataset. Data exploration folder contains answers of some questions such as:
- what tables are available and what information do they contain?
- How are the tables related?
- Are there duplicate of missing values?
- What are the primary keys and candidate foreign keys?
- How is the data distributed across customers, orders, products, and sellers?

The exploration is organized into the following sections:
| File  | Description  |
| '01_data_quality.md' | Missing values, duplicates, primary keys, and data validation |
| '02_customer_analysis.md' | Exploration of customer information and geographic distribution |
| '03_order_analysis.md' | Order statuses, dates, delivery information, and order characteristics |
| '04_product_analysis.md' | Product categories, pricing, and product-related exploration |
| '05_seller_analysis.md' | Seller distribution and basic seller statistics |

The purpose of this phase is to gain a solid understanding of the dataset before moving on to business-focused analysis and advanced SQL queries.
